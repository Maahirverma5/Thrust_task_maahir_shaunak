Bubble Popper:
A webcam-based arcade game using OpenCV and MediaPipe. Slice bubbles using hand gestures, featuring 2-Player Co-op as well.

to install dependencies run the command: 
        pip install opencv-python mediapipe pygame

How to play: 
Run the game: python main.py

Menu: slice your finger over the buttons to select 1 Player or 2 Player mode.

Goal: Slice colored bubbles. Avoid black bombs.

Controls: 
        Q - quit
        R- restart
        M - Retrun to menu

Have fun Slicing!!!
